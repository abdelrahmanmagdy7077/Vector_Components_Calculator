# Vector Component Calculator (No Math Libraries)

This is a quick project I put together to experiment and apply some of the concepts I picked up during my first year of engineering. 

I wanted to see how easy it would be to build the standard tools I usually just import. Instead of using Python's `math` library, I built my own `sin` and `cos` functions from the ground up using Taylor series expansions and trig identities. I added some basic input checks and used it all to calculate the x and y components of physics vectors.

Just a fun, simple exercise in connecting the math I learned in uni to actual code.
